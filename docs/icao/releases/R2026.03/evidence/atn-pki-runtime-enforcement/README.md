# ATN PKI runtime enforcement assurance artifacts (R2026.03)

Artifacts in this folder are produced by:

```bash
scripts/evidence/atn_pki_runtime_enforcement_assurance.sh R2026.03
```

Per run, the script publishes timestamped artifacts:

- `<timestamp>-execution.log`
- `<timestamp>-configuration-snapshot.txt`
- `<timestamp>-verdict.md`
- `<timestamp>-manifest.txt`

`latest-manifest.txt` points to the newest timestamped manifest.
